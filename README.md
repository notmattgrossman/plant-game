# Plant Game

An interactive plant watering game built with p5.js. Water plants by hovering over them with your mouse to watch them grow and bloom into beautiful flowers!

## Features

- **Interactive Watering**: Move your mouse around to water plants with a realistic watering can
- **Plant Growth Animation**: Watch plants grow from seeds to fully bloomed flowers over 5 seconds
- **Multiple Flower Types**: Three different flower varieties - sunflowers, pink flowers, and blue flowers
- **Visual Effects**: Animated water droplets and smooth watering can rotation
- **Progress Tracking**: Real-time flower count and game timer
- **Beautiful Graphics**: Custom SVG artwork for plants, flowers, and background

## How to Play

1. Open `index.html` in a web browser
2. Move your mouse around the screen - the watering can follows your cursor
3. Hover over the pots to water the plants
4. Watch as the plants grow stems and bloom into flowers
5. Try to grow all 9 flowers!

## Game Details

- **Total Plants**: 9 plants arranged in a grid pattern
- **Growth Time**: Each plant takes 5 seconds to fully grow when watered
- **Flower Types**: 
  - Sunflowers (yellow)
  - Pink flowers
  - Blue flowers (larger size)
- **Layout**: 3 rows with the middle row offset for visual interest

## Controls

- **Mouse Movement**: Controls the watering can position
- **Hover over pots**: Automatically waters plants and triggers growth

## Technologies Used

- p5.js - Creative coding library
- HTML5 Canvas
- SVG graphics for scalable plant and flower artwork

## Files

- `index.html` - Main HTML file
- `sketch.js` - Game logic and p5.js code
- `style.css` - Styling
- `img/` - Image assets (background, pot, stem, and flower SVGs)
