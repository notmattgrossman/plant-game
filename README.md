# p5.js Processing Code

A comprehensive p5.js example demonstrating various processing techniques including:

- **Particle Systems**: Dynamic particle generation and physics simulation
- **Data Visualization**: Real-time data point rendering with connections
- **Image Processing**: Multiple filter modes (blur, sharpen, edge detection)
- **Noise Processing**: Perlin noise for organic movement
- **Interactive Graphics**: Mouse and keyboard controls

## Features

- Real-time particle system with physics
- Data point visualization with dynamic connections
- Multiple visual filter modes
- Grid overlay option
- Interactive controls

## Controls

- **1**: No filter
- **2**: Blur filter
- **3**: Sharpen filter
- **4**: Edge detection filter
- **G**: Toggle grid
- **R**: Reset particles and data
- **Mouse Click**: Add new particle and data point

## How to Run

1. Open `index.html` in a web browser
2. The sketch will automatically start running
3. Use keyboard and mouse controls to interact

## Processing Techniques Demonstrated

1. **Data Processing**: Sorting, mapping, and transforming data points
2. **Signal Processing**: Noise-based movement and filtering
3. **Image Processing**: Pixel manipulation and filter effects
4. **Real-time Processing**: Continuous updates and animation
5. **Interactive Processing**: User input handling and response
